# 2022-C 2022年 CUMCM C题：古代玻璃制品的成分分析与鉴别

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2022年 CUMCM C题：古代玻璃制品的成分分析与鉴别` 的业务场景转化为可计算的数学模型，重点围绕 机器学习与统计、评价与决策模型、微分方程与动力系统 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：对这些玻璃文物的表面风化与其玻璃类型、纹饰和颜色的关系进行分析；结合玻 璃的类型，分析文物样品表面有无风化化学成分含量的统计规律，并根据风化点检测数据，预 测其风化前的化学成分含量。
