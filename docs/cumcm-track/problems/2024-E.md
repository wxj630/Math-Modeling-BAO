# 2024-E 2024年 CUMCM E题：交通流量管控

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2024年 CUMCM E题：交通流量管控` 的业务场景转化为可计算的数学模型，重点围绕 图论与复杂网络、评价与决策模型、规划优化模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：对经中路-纬中路交叉口，根据车流量的差异，可将一天分成若干个时段，估 计不同时段各个相位（包括四个方向直行、转弯）车流量。
