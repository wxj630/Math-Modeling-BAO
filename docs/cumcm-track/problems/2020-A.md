# 2020-A 2020年 CUMCM A题：炉温曲线

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2020年 CUMCM A题：炉温曲线` 的业务场景转化为可计算的数学模型，重点围绕 微分方程与动力系统、数据处理与拟合、规划优化模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：请对焊接区域的温度变化规律建立数学模型。假设传送带过炉速度为78 cm/min，各温区温度的设定值分别为173ºC（小温区1~5）、198ºC（小温区6）、230ºC（小温区7）和257ºC（小温区8~9），请给出焊接区域中心的温度变化情况

### q01 问题1

**递进作用：** 建立整题主模型或数据入口，后续小问通常都继承这里的变量、指标或数据清洗结果。

**题意摘要：** 请对焊接区域的温度变化规律建立数学模型。假设传送带过炉速度为78 cm/min，各温区温度的设定值分别为173ºC（小温区1~5）、198ºC（小温区6）、230ºC（小温区7）和257ºC（小温区8~9），请给出焊接区域中心的温度变化情况，列出小温区3、6、7中点及小温区8结束处焊接区域中心的温度，画出相应的炉温曲线，并将每隔0.5 s焊接区域中心的温度存放在提供的result.csv中。
