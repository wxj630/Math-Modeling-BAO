# 2021-C 2021年 CUMCM C题：生产企业原材料的订购与运输

> 这是一个赛题整体入口。先看整题主线，再进入 6 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2021年 CUMCM C题：生产企业原材料的订购与运输` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、数据处理与拟合、评价与决策模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：根据附件1，对402 家供应商的供货特征进行量化分析，建立反映保障企业生产 重要性的数学模型，在此基础上确定50 家最重要的供应商， 并在论文中列表给出结果。
