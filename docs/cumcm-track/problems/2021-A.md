# 2021-A 2021年 CUMCM A题：FAST”主动反射面的形状调节

> 这是一个赛题整体入口。先看整题主线，再进入 11 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2021年 CUMCM A题：FAST”主动反射面的形状调节` 的业务场景转化为可计算的数学模型，重点围绕 几何与方程模型、评价与决策模型、微分方程与动力系统 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：当待观测天体𝑆位于基准球面正上方，即𝛼 = 0°, 𝛽 = 90°时，结合考虑反射面板调节因 素，确定理想抛物面。
