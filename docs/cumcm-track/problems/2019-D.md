## 整题主线

本题核心是把 `2019年 CUMCM D题：空气质量数据的校准` 的业务场景转化为可计算的数学模型，重点围绕 数据处理与拟合、微分方程与动力系统、机器学习与统计 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：对自建点数据与国控点数据进行探索性数据分析。

## Baseline 全局报告

Baseline 层把整题拆成 3 个最低可运行脚手架，覆盖的通用模型族为：quadratic_least_squares×3。它的价值不是给出最终答案，而是快速回答三个问题：题面有哪些可用数字，应该从哪个经典模型族切入，代码、结果和报告是否能跑通。
