# 2017-C 2017年 CUMCM C题：颜色与物质浓度辨识

> 这是一个赛题整体入口。先看整题主线，再进入 2 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2017年 CUMCM C题：颜色与物质浓度辨识` 的业务场景转化为可计算的数学模型，重点围绕 数据处理与拟合、评价与决策模型、机器学习与统计 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：附件Data1.xls中分别给出了5种物质在不同浓度下的颜色读数，讨论从这5组数据中能否确定颜色读数和物质浓度之间的关系，并给出一些准则来评价这5组数据的优劣
