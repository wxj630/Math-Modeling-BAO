# 2018-B 2018年 CUMCM B题：赛题

> 这是一个赛题整体入口。先看整题主线，再进入 3 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2018年 CUMCM B题：赛题` 的业务场景转化为可计算的数学模型，重点围绕 几何与方程模型、微分方程与动力系统、规划优化模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：一道工序的物料加工作业情况，每台CNC安装同样的刀具，物料可以在任一台CNC上加工完成；
