# 2011-C 2011年 CUMCM C题：企业退休职工养老金制度的改革

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2011年 CUMCM C题：企业退休职工养老金制度的改革` 的业务场景转化为可计算的数学模型，重点围绕 数据处理与拟合、时间序列模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：对未来中国经济发展和工资增长的形势做出你认为是简化、合理的假设，并参考附件1，预测从2011年至2035年的山东省职工的年平均工资。
