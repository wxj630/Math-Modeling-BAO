# 2019-C 2019年 CUMCM C题：赛题

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2019年 CUMCM C题：赛题` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、评价与决策模型、微分方程与动力系统 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：分析研究与出租车司机决策相关因素的影响机理，综合考虑机场乘客数量的变化规律和出租车司机的收益，建立出租车司机选择决策模型，并给出司机的选择策略。
