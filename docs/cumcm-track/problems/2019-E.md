# 2019-E 2019年 CUMCM E题：薄利多销”分析

> 这是一个赛题整体入口。先看整题主线，再进入 3 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2019年 CUMCM E题：薄利多销”分析` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、数据处理与拟合、评价与决策模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：计算该商场从2016年11月30日到2019年1月2日每天的营业额和利润率（注意：由于未知原因，数据中非打折商品的成本价缺失。一般情况下，零售商的利润率在20%-40%之间)
