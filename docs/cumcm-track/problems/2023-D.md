# 2023-D 2023年 CUMCM D题：圈养湖羊的空间利用率

> 这是一个赛题整体入口。先看整题主线，再进入 3 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2023年 CUMCM D题：圈养湖羊的空间利用率` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、时间序列模型、几何与方程模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：不考虑不确定因素和种羊的淘汰更新，假定自然交配期 20 天，母羊都能受孕，孕 期 149 天，每胎产羔 2 只，哺乳期 40 天，羔羊育肥期 210 天，母羊空怀休整期 20 天。该湖羊 养殖场现有 112 个标准羊栏，在实现连续生产的条件…
