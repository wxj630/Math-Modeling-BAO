# 2014-D 2014年 CUMCM D题：储药柜的设计

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2014年 CUMCM D题：储药柜的设计` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、数据处理与拟合 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：药房内的盒装药品种类繁多，药盒尺寸规格差异较大，附件1中给出了一些药盒的规格。请利用附件1的数据，给出竖向隔板间距类型最少的储药柜设计方案，包括类型的数量和每种类型所对应的药盒规格。

### q02 问题 2

**递进作用：** 在前问基础上加入新的约束、数据或评价口径，推进整题模型链。

**题意摘要：** 药盒与两侧竖向隔板之间的间隙超出2mm的部分可视为宽度冗余。增加竖向隔板的间距类型数量可以有效地减少宽度冗余，但会增加储药柜的加工成本，同时降低了储药槽的适应能力。设计时希望总宽度冗余尽可能小，同时也希望间距的类型数量尽可能少。仍利用附件1的数据，给出合理的竖向隔板间距类型的数量以及每种类型对应的药品编号。

### q03 问题 3

**递进作用：** 在前问基础上加入新的约束、数据或评价口径，推进整题模型链。

**题意摘要：** 考虑补药的便利性，储药柜的宽度不超过2.5m、高度不超过2m，传送装置占用的高度为0.5m，即储药柜的最大允许有效高度为1.5m。药盒与两层横向隔板之间的间隙超出2mm的部分可视为高度冗余，平面冗余＝高度冗余×宽度冗余。在问题2计算结果的基础上，确定储药柜横向隔板间距的类型数量，使得储药柜的总平面冗余量尽可能地小，且横向隔板间距的类型数量也尽可能地少。
