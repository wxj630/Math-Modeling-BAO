# 2014-A 2014年 CUMCM A题：嫦娥三号软着陆轨道设计与控制策略

> 这是一个赛题整体入口。先看整题主线，再进入 3 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2014年 CUMCM A题：嫦娥三号软着陆轨道设计与控制策略` 的业务场景转化为可计算的数学模型，重点围绕 几何与方程模型、规划优化模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：确定着陆准备轨道近月点和远月点的位置，以及嫦娥三号相应速度的大小与方向。
