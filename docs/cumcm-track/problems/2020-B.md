# 2020-B 2020年 CUMCM B题：穿越沙漠

> 这是一个赛题整体入口。先看整题主线，再进入 13 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2020年 CUMCM B题：穿越沙漠` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、时间序列模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：以天为基本时间单位，游戏的开始时间为第0天，玩家位于起点。玩家必须在截止日期或之前到达终点，到达终点后该玩家的游戏结束。
