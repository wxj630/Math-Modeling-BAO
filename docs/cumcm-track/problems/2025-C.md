# 2025-C 2025年 CUMCM C题：NIPT 的时点选择与胎儿的异常判定

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2025年 CUMCM C题：NIPT 的时点选择与胎儿的异常判定` 的业务场景转化为可计算的数学模型，重点围绕 评价与决策模型、机器学习与统计、数据处理与拟合 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：试分析胎儿Y 染色体浓度与孕妇的孕周数和BMI 等指标的相关特性，给出相应的关系模 型，并检验其显著性。
