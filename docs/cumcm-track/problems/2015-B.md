# 2015-B 2015年 CUMCM B题：互联网+”时代的出租车资源配置

> 这是一个赛题整体入口。先看整题主线，再进入 3 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2015年 CUMCM B题：互联网+”时代的出租车资源配置` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、评价与决策模型、图论与复杂网络 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：试建立合理的指标，并分析不同时空出租车资源的“供求匹配”程度。
