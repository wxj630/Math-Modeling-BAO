# 2019-B 2019年 CUMCM B题：同心协力”策略研究

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2019年 CUMCM B题：同心协力”策略研究` 的业务场景转化为可计算的数学模型，重点围绕 几何与方程模型、评价与决策模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：在理想状态下，每个人都可以精确控制用力方向、时机和力度，试讨论 这种情形下团队的最佳协作策略，并给出该策略下的颠球高度。
