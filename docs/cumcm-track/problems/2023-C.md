## 整题主线

本题核心是把 `2023年 CUMCM C题：蔬菜类商品的自动定价与补货决策` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、几何与方程模型、数据处理与拟合 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：蔬菜类商品不同品类或不同单品之间可能存在一定的关联关系，请分析蔬菜各 品类及单品销售量的分布规律及相互关系。
