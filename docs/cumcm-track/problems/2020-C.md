# 2020-C 2020年 CUMCM C题：中小微企业的信贷决策

> 这是一个赛题整体入口。先看整题主线，再进入 10 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2020年 CUMCM C题：中小微企业的信贷决策` 的业务场景转化为可计算的数学模型，重点围绕 评价与决策模型、规划优化模型、数据处理与拟合 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：对附件1中123家企业的信贷风险进行量化分析，给出该银行在年度信贷总额固定时对这些企业的信贷策略。
