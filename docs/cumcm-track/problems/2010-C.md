# 2010-C 2010年 CUMCM C题：输油管的布置

> 这是一个赛题整体入口。先看整题主线，再进入 3 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2010年 CUMCM C题：输油管的布置` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：针对两炼油厂到铁路线距离和两炼油厂间距离的各种不同情形，提出你的设计方案。在方案设计时，若有共用管线，应考虑共用管线费用与非共用管线费用相同或不同的情形。
