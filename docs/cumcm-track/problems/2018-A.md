# 2018-A 2018年 CUMCM A题：高温作业专用服装设计

> 这是一个赛题整体入口。先看整题主线，再进入 2 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题核心是把 `2018年 CUMCM A题：高温作业专用服装设计` 的业务场景转化为可计算的数学模型，重点围绕 规划优化模型、微分方程与动力系统 展开；起手应先梳理变量、约束、目标函数/评价指标，再按每问逐步建模。第一问通常给出主要建模入口：当环境温度为65ºC、IV层的厚度为5.5 mm时，确定II层的最优厚度，确保工作60分钟时，假人皮肤外侧温度不超过47ºC，且超过44ºC的时间不超过5分钟。

- 问题：当环境温度为65ºC、IV层的厚度为5.5 mm时，确定II层的最优厚度，确保工作60分钟时，假人皮肤外侧温度不超过47ºC，且超过44ºC的时间不超过5分钟。

- 分析：baseline 适合作为建模起点和覆盖检查；它还没有充分吸收题目专用数据、业务约束和论文表达要求。
