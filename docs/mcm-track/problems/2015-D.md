# 2015-D ICM-D: Is it sustainable?

> 这是一个赛题整体入口。先看整题主线，再进入 4 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题共 4 个小问。阅读时不要把小问拆成孤岛：`国家可持续性评价模型` 通常给出主模型或数据入口，后续小问逐步加入动态、情景、评价、决策或论文表达要求，最后由 `20 页报告、模型优缺点与 ICM 投资建议` 收束成整题结论。
