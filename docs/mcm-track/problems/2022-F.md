# 2022-F ICM-F: All for One and One (Space) for All!

> 这是一个赛题整体入口。先看整题主线，再进入 8 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题共 8 个小问。阅读时不要把小问拆成孤岛：`外层空间条约承诺与全球公平定义` 通常给出主模型或数据入口，后续小问逐步加入动态、情景、评价、决策或论文表达要求，最后由 `Outer Space Treaty 更新的一页 UN 备忘录` 收束成整题结论。
