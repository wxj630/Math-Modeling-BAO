# 2019-A MCM-A: Game of Ecology

> 这是一个赛题整体入口。先看整题主线，再进入 7 个小问的 baseline、advanced 和 outstanding 预留位。

## 整题主线

本题共 7 个小问。阅读时不要把小问拆成孤岛：`龙的生态影响与栖息要求` 通常给出主模型或数据入口，后续小问逐步加入动态、情景、评价、决策或论文表达要求，最后由 `给 George R.R. Martin 的信与现实类比` 收束成整题结论。
