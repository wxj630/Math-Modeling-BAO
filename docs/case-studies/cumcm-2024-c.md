# CUMCM 2024-C 农作物规划案例

2024-C 是 CUMCM 中非常典型的整题递进：先在稳定参数下求 2024-2030 种植方案，再加入不确定性，最后考虑作物替代、互补和相关性。

赛题页入口：[2024-C 农作物的种植策略](/cumcm-track/problems/2024-C)

## 小问递进链

| 小问 | 在整题中的作用 | Advanced 重点 |
|---|---|---|
| q01 | 建立稳定参数下的主规划模型。 | 地块、季次、作物、销售上限、滞销/折价两种收益规则。 |
| q02 | 在 q01 基础上加入未来不确定性。 | 销量增长/波动、亩产波动、成本上涨、价格变化和风险。 |
| q03 | 在 q02 基础上考虑相关结构。 | 替代性、互补性、销售量与价格/成本相关性、模拟比较。 |

## Baseline 到 advanced 的变化

Baseline 使用 `linear_programming` 保留规划入口：

```text
max sum_i c_i * x_i
s.t. A * x <= b, x >= 0
```

Advanced 把变量专门化为：

```text
x_{y,l,s,c}: 第 y 年地块 l 在季次 s 种植作物 c 的面积
Y_{l,c,s}: 地块类型-作物-季次亩产量
C_{l,c,s}: 种植成本
P_{c,s}: 销售单价区间中点
D_{y,c,s}: 预期销售量上限
```

## 为什么要按整题读

如果只看 q01，会以为这只是一个线性规划题；但 q02 和 q03 会不断改写 q01 的参数假设和收益结构。真正的论文主线应该是：

1. 稳定情景下的基础种植方案。
2. 面向未来波动的风险调整方案。
3. 考虑作物相关性的稳健策略比较。

## Outstanding 预留

- 用混合整数规划或更系统的元启发式替代贪心搜索。
- 对销量、亩产、成本、价格做相关 Monte Carlo 和稳健优化。
- 生成年度利润曲线、作物结构图和地块热力图。
- 把结果解释写成“策略建议 + 风险提示 + 管理启示”的完整章节。
